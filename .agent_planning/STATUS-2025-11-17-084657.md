```markdown
## Refinement Mode

The `--refine` flag runs a second pass with 2x parallelism to improve naming quality.

**How it works**:
1. Pass 1: Processes original file with webcrack → plugins → prettier
2. Webcrack extracts to N files (bundle_1.js, bundle_2.js, etc.)
3. Pass 2: Re-processes each extracted file (skips webcrack) with 2x concurrency
4. Result: Higher quality names at 2x cost

**Example**:
```bash
humanify unminify --provider openai large-bundle.js --refine --turbo
```

**Note**: Only supported for OpenAI provider. Gemini and Local LLM coming soon.
